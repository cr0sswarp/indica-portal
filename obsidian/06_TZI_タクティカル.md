## 🏗️ TZI システム概要

**目的**: 試合映像 → 選手軌跡データ → 戦術インサイト → ハルへの言語化

```
試合映像 (Google Drive)
  → OpenCV フレーム抽出
  → Homography ピクセル→フィールド座標変換
  → JSON 軌跡データ
  → matplotlib ヒートマップ
  → 戦術言語化（このノート）
  → ハルへのLINEメッセージ（かれん原理）
```
