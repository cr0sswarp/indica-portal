## ⚽ 戦術的役割マップ（ポリバレント設計）

### モード1：**プレッシャー起点 × ショートカウンター**（基本形）
```
ハルがDFライン前で相手を積極追い込み
  ↓
① ボール回収 → ショートカウンター（形A: スルーパス）
② ボール回収 → ショートカウンター（形B: 自分で得点を取りに行く）
③ 相手のプレッシャーによるミス誘発（間接効果）
  ↓
→ アンカー(#10)とDFラインへの負担が激減
→ 攻撃時間の比率が圧倒的に高まる
```

### モード2：**2枚アンカー**（守備強化モード）
```
守備が必要な展開
  ↓
ハル ＋ #10 で役割を共有
→ 中盤に2枚のアンカーが形成される
→ 強固な守備ブロック ＋ カウンター準備態勢
```

### モード3：**ハルが出し手 × #10がターゲット**（高さ不要時）
```
DFラインに大きな2人がいる → 身長差は問題なし
  ↓
#10（もともとフォワード）を自由に前に走らせる
ハル = ボールの出し手（配給役）に転換
  ↓
① ターゲットとしての#10へのパス → 抜け出しに合わせる
② センタリング供給（#10を飛び出す的として活用）
③ 林の縦の差し込み ＋ ハルの右足による深いところからの縦のスイッチ
  ↓
→ 完全にセントラルミッドフィルダーとして
  守備の基点 ＋ 攻撃の起点 の両面を担う
```

## 📈 育成ロードマップ

### **ステージ1（今ここ）: ボールを失わない選手へ**
> 「使えない選手から完全脱却する」

- **技術**: プレッシャー下でのボールコントロール
- **体力**: 90分間プレス強度を維持できるフィジカル
- **マインドセット**: ミスへの恐怖を取り除く → 0550状態での出場

**具体的課題:**
- [ ] プレッシャー中に選択肢を2つ以上持った状態でボールを受ける
- [ ] 受けた瞬間に「出し手の優先順位」を体に染み込ませる

## 🧠 マインドセット：0550 原則

> **常に0550となるような状態を試合前に完全準備する**

- すっきり × ワクワク した状態で試合に入る
- 不安・迷い・緊張ではなく「今日何かが起こる」という期待感
- 試合前の準備（睡眠・食事・ウォームアップ・思考）で0550を作り出す

**0550チェックリスト（試合前）:**
- [ ] 十分な睡眠が取れているか
- [ ] 頭が「すっきり」しているか（雑念がないか）
- [ ] 「ワクワク」しているか（楽しみにしているか）
- [ ] 今日の自分の役割を1つの核心で言語化できるか
