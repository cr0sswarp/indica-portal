# 動画確認・修正の手順（牧野羽瑠向け）

## やること（3ステップだけ）

```
1. Google Drive > TZI > overlays の動画を見る
2. Drive > TZI > corrections > フォルダを作って corrections.txt を書く
3. 終わったら連絡するだけ
```

---

## ステップ1: 動画を見る

**Google Drive > TZI > overlays** に各試合のオーバーレイ動画がある。

```
overlay_20260325_vs立教大_03-25.mp4
overlay_20260314_vs埼玉大_03-14.mp4
...
```

再生すると青い枠で選手が囲われ、**シアン（水色）の下向き矢印**が自分（羽瑠）を追跡している。

---

## ステップ2: 修正を書く

**Drive > TZI > corrections** に試合名のフォルダを作る。

```
TZI/
└── corrections/
    └── 20260325_vs立教大/        ← このフォルダを作る
            corrections.txt       ← この中にテキストファイルを作る
```

フォルダ名の先頭8桁（`20260325` の部分）だけ正確であればOK。

---

## corrections.txt の書き方

**ルールなし。気づいたことを日本語で書くだけ。**

以下はすべて書いてもよい例：

```
1H 10分 OK
1H 15分 NG 矢印が隣の選手を指している。自分は左にいた。
1H 30分 OK
前半は全体的に合ってる気がする

2H 5分 OK
2H 29分30秒からボランチに上がった（右ボランチ）
2H 35分 NG 誰かわからない人を指している
2H 40分 OK

後半の方が精度高いと思う
16番と13番を混同しているかも
```

---

## 書き方のポイント

| 書きたいこと | 書き方の例 |
|---|---|
| 矢印が正しい | `1H 10分 OK` または `10分 合ってる` |
| 矢印が間違い | `1H 15分 NG` または `15分 違う人を指している` |
| ポジション変化 | `2H 29分30秒からボランチ` または `後半30分〜右SBに戻った` |
| 背番号の間違い | `1H 20分 #14が#6と間違えている` |
| 全体コメント | そのまま文章で書く |

---

## 書かなくていいこと

- 全フレームを確認しなくてもOK
- 1試合の中で気づいた箇所だけでも十分
- 「よくわからなかった」もそのまま書いてOK

---

## 完了したら

LINE または任意の連絡手段で **「corrections.txt 置いた」** と一言。

Claude 側が自動で内容を読んで JSON 化し、次回の分析精度を上げます。

---

## Drive フォルダ構成（参考）

```
TZI/
├── videos/       試合動画（元々あるもの）
├── overlays/     追跡オーバーレイ動画（Claude がアップロード）
└── corrections/
    ├── 20260325_vs立教大/
    │   ├── corrections.txt        ← 羽瑠が書く
    │   └── corrections_parsed.json ← Claude が自動生成（参考用）
    └── 20260314_vs埼玉大/
        └── corrections.txt
```
