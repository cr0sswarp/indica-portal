# 牧野羽瑠へ — 動画確認・修正の手順

## 流れ

```
1. Google Drive に動画（または GIF）が届く
         ↓
2. 動画を見ながら、このガイドに従って修正ファイルを書く
         ↓
3. ファイルを Google Drive の TZI/corrections/ に置く
         ↓
4. Claude 側が自動で読み取って精度を上げる
```

---

## ステップ1: ファイルをコピーする

Google Drive の **TZI/corrections/** フォルダを開く。

`TEMPLATE_corrections.json` をコピーして、試合の日付でリネーム：

```
corrections_20260325.json   ← vs立教大
corrections_20260314.json   ← vs埼玉大
```

---

## ステップ2: 動画を見ながら修正を記入する

### 書き方の例

```json
{
  "match": "20260325",
  "reviewed_at": "2026-05-25",
  "reviewer": "牧野羽瑠",
  "notes": "全体的にかなり良くなっている。後半の方が精度高め。",

  "arrow_checks": [
    { "time": "1H 1min",  "status": "ok" },
    { "time": "1H 15min", "status": "ok" },
    { "time": "1H 30min", "status": "wrong", "note": "矢印が隣の選手を指している。自分は画面左側にいた" },
    { "time": "2H 15min", "status": "ok" },
    { "time": "2H 30min", "status": "ok" }
  ],

  "position_fixes": [
    {
      "jersey": 6,
      "half": "2H",
      "from_min": 29.5,
      "to_min": 999,
      "role": "セントラル",
      "label": "右ボランチ",
      "note": "29分30秒から中盤に上がった"
    }
  ],

  "jersey_fixes": [],

  "free_comments": [
    "前半は右SBの位置にいた",
    "16番と13番の区別がまだできていないかも"
  ]
}
```

---

## 記入ルール

### arrow_checks（矢印チェック — 最重要）

| フィールド | 値 | 説明 |
|---|---|---|
| `time` | `"1H 15min"` | 前半/後半と分数 |
| `status` | `"ok"` | 矢印が自分を正しく指している |
| `status` | `"wrong"` | 矢印が間違った人を指している |
| `note` | 任意 | どこにいたか、何が違うか |

### position_fixes（ポジション修正）

| フィールド | 例 | 説明 |
|---|---|---|
| `half` | `"1H"` / `"2H"` | 前半/後半 |
| `from_min` | `29.5` | 開始分 (29分30秒 = 29.5) |
| `to_min` | `999` | 終了分 (試合終了まで = 999) |
| `role` | 下の表を参照 | ポジション名 |
| `label` | `"右SB"` | 表示用の名前（自由） |

**role の選択肢:**
- `"サイドバック"` — SB
- `"アンカー"` — ボランチ（守備的）
- `"セントラル"` — ボランチ（攻撃的）/ インサイドハーフ
- `"トップ下"` — トップ下
- `"トップ"` — FW
- `"センターバック"` — CB

### jersey_fixes（背番号の修正）

矢印が明らかに違う選手を指していた場合に記入。

```json
{ "time": "1H 20min", "correct_jersey": 14, "note": "矢印は#14を指していた" }
```

---

## 記入しなくていいこと

- 全フレームをチェックしなくてもOK
- 分からない時刻は空欄でOK
- `_example` がついている行はそのまま残してOK（自動でスキップされる）

---

## ファイルを置く場所

**Google Drive > TZI > corrections > corrections_20260325.json**

置いたら LINE か何かで牧野さんに「置いたよ」と一言送れば Claude 側が同期します。

---

## どんな修正でも歓迎

- 「この時刻は当たってる」という確認だけでも価値あり
- 「全然違う」というザックリなコメントも OK
- 試合全体を通して気になった傾向（例：「前半は全体的にズレている」）も自由欄に
